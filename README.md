# EID-Algebra-Lineal
Espacio para desarrollo del EID de Algebra Lineal.

Leer instrucciones en el PDF y implementar los criterios solicitados (Instalar la extensión 'vscode-pdf' para leer el PDF directamente
desde VSC en caso de no tenerlo por si les acomoda más).

Trabajar en su rama asignada para ir implementando nuevas funcionalidades o nuevos modulos, una vez implementadas y verificar que
funcionan correctamente las nuevas funcionalidades hacer el commit a su rama y luego subir los nuevos cambios en la rama main para que el resto de integrantes pueda trabajar con los ultimos cambios (Idealmente informar por el chat de Discord para que cada integrante este al tanto de quien y que implementaciones/cambios han sido incorporados al código.)

En caso de implementar alguna nueva libreria para utilizar en el proyecto añadirla al archivo 'requirements.txt' para que el resto
pueda estar al tanto e instalarla de forma más sencilla.

Ideal tener código/módulos lo más organizados posible para tener un orden de este.

Recordar el tener cuidado de copia directa de codigo de la IA ya que según instrucciones puede llegar a ser gravemente sancionable.

Cualquier pregunta/duda acerca del código no duden en consultar en el chat de Discord para que podamos resolverlas.

Ideal estar en constante comunicación con el resto de integrantes en el desarrollo del proyecto para tener un buen avance en este.
